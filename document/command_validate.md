
# --validate
- 접속이 가능한 상태인지 접속정보를 체크합니다
    ```
    j18 --validate
    ```

- 실패시 다음 3 가지중 하나의 [error code](error_code.md)가 출력될 수 있습니다.
    - [error:0001] connection failed
    - [error:0002] Permission declined(error_code.md)
    - [error:0003] Repos not found(error_code.md)
