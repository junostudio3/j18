# j18 v0.0.0.8
- Prompt 에서 Seafile 클라우드에 명령을 내리기 위한 프로그램입니다.
- 매크로 등을 사용하여 편리하게 자동화에 사용할 만한 프로그램을 개발하는게 목표입니다.

## [처음 시작하기](get_started_with_j18.md)

## [config](config.md)

## format
- j18은 다음의 형식으로 명령을 지시합니다.
    ```
    j18 <command> [options]
    ```
